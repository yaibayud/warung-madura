# 🏪 Warung Madura Modern

[![JavaScript](https://img.shields.io/badge/javascript-%23323330.svg?style=for-the-badge&logo=javascript&logoColor=%23F7DF1E)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
[![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)](https://developer.mozilla.org/en-US/docs/Web/CSS)

Warung Madura Modern adalah solusi web e-commerce sederhana dan sistem manajemen inventaris (Mini POS) yang dirancang khusus untuk kebutuhan warung kelontong tradisional dengan sentuhan estetika modern. Aplikasi ini mengusung tagline ikonik **"Buka 24 Jam"** dan palet warna khas Warung Madura (Merah, Kuning, Hijau).

## ✨ Fitur Utama

### 🛒 Landing Page Pelanggan
- **Hero Section Estetik**: Tampilan visual yang menarik dengan branding yang kuat.
- **Katalog Produk**: Grid produk responsif dengan filter kategori (Sembako, Rokok, Minuman, Sabun, Mie Instan).
- **Pencarian Cepat**: Cari produk secara instan melalui kolom pencarian.
- **Indikator Stok**: Label status real-time (Stok Tersedia, Hampir Habis, Stok Habis).
- **Simulasi Transaksi**: Notifikasi interaktif saat menambahkan produk ke keranjang.

### 📊 Dashboard Admin
- **Ringkasan Statistik**: Pantau total produk, peringatan stok rendah, dan omzet harian.
- **Manajemen Inventaris**: Tabel pengelolaan stok yang intuitif dengan fitur edit dan hapus.
- **Responsif**: Dashboard yang nyaman digunakan baik di desktop maupun tablet.

## 🚀 Teknologi yang Digunakan

- **Core**: HTML5, Vanilla JavaScript.
- **Styling**: Vanilla CSS dengan desain *Glassmorphism* dan *Premium UI*.
- **Icons**: [Lucide Icons](https://lucide.dev/).
- **Fonts**: [Outfit](https://fonts.google.com/specimen/Outfit) via Google Fonts.
- **Build Tool**: [Vite](https://vitejs.dev/).

## 🛠️ Cara Menjalankan Proyek

1. **Clone repositori ini:**
   ```bash
   git clone https://github.com/yaibayud/warung-madura.git
   ```

2. **Masuk ke direktori proyek:**
   ```bash
   cd warung-madura
   ```

3. **Install dependensi:**
   ```bash
   npm install
   ```

4. **Jalankan server pengembangan:**
   ```bash
   npm run dev
   ```

5. **Akses aplikasi:**
   - Landing Page: `http://localhost:5173/`
   - Admin Dashboard: `http://localhost:5173/admin.html`
